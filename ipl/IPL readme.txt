# IPL Performance Analysis Dashboard

## Overview
An interactive Power BI dashboard built to analyze IPL team and player performance metrics.

## Tools Used
- Power BI
- Excel

## Dashboard Features
- Team-wise performance analysis
- Player statistics tracking
- Runs and wickets analysis
- Match performance insights
- KPI cards and interactive filters

## Key Insights
- Compared team performances across matches.
- Analyzed player contributions using batting and bowling statistics.
- Identified performance trends through interactive visualizations.

## Skills Demonstrated
- Data Visualization
- Dashboard Development
- KPI Reporting
- Sports Analytics
- Business Intelligence
